# My Mission Control (Next.js)

個人用のミッションコントロールを Next.js で作成。

## 入っている3ツール

1. **Quick Capture Inbox**
   - 思いついたタスクを即登録
   - 完了チェック付き（localStorage保存）

2. **Focus Sprint Timer**
   - 15/25/45分プリセットの集中タイマー
   - 開始/停止/リセット

3. **Bottleneck Radar**
   - 工程ごとの所要時間を記録
   - 平均サイクルタイムと遅い工程 Top3 を表示

## 起動

```bash
npm install
npm run dev
```

ブラウザで `http://localhost:3000` を開く。
